# Dog-Friendly Places Finder

## Project Overview
Dog-Friendly Places Finder is a Java object-oriented application that helps dog owners find places where dogs are allowed. Users can search for dog-friendly places, filter places by features, view place details, and add new dog-friendly place information.
This project was created for an object-oriented programming assignment. It includes Java classes, UML diagrams, documentation, a console version, and a Java Swing graphical user interface.

## Features
- Search dog-friendly places by keyword
- Filter places by features such as water access, shade, outdoor seating, and off-leash areas
- View details for a selected place
- Add new dog-friendly places
- View ratings and reviews
- Run a graphical user interface using Java Swing

## Java Concepts Used
- Classes and objects
- Encapsulation
- Constructors
- Methods
- ArrayLists
- JavaDoc comments
- Preconditions and postconditions
- Object relationships
- GUI event handling with Java Swing

## Project Files
- `Main.java` - Runs the console version of the program
- `DogFriendlyPlacesGUI.java` - Runs the graphical user interface
- `DogFriendlyPlacesFinder.java` - Main controller class for searching, filtering, and managing places
- `DogFriendlyPlace.java` - Represents a dog-friendly location
- `DogOwner.java` - Represents the dog owner user
- `BusinessOwner.java` - Represents the business owner user
- `PlaceFeature.java` - Represents dog-friendly features
- `Review.java` - Represents reviews and ratings

## How to Run

To Compile Project:
```bash
javac *.java
```

To run console:
```bash
java Main
```

To run the GUI:
```bash
java DogFriendlyPlacesGUI
```
